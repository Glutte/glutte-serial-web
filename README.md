How to setup
============

 1. Install virtualenv
 2. Clone this repo
 3. Setup a new virtualenv
 4. Copy config.py.dist to config.py. Edit variables inside


Step-by-step
------------

    cd glutte-serial-web
    virtualenv .
    source bin/activate
    pip install -r requirements.txt
    cp config.py.dist config.py
    vim config.py
